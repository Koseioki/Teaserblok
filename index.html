Notes

  • This pattern uses <a> only for the headings, rather than for wrapping the whole block.
  • The whole block <div> is clickable by using onClick=....
  • When the block is focused , it gets outline and the heading inside gets underline.
  • Screen readers will understand this component as just a heading with a link, and text.
  • A black gradient background is added to make sure that there is enough colour contrast
